# rotatorCuffRecovery

TinyML device to recover from rotator cuff injury 

Parts:
Arduino Nano BLE 33 Sense
Oled screen
Switch
3.7 battery
TP4056 charger

Add the zip file to Arduino Ide, Sketch, include library, Add Zip 
Then open rotatorCuff8.ino and upload to Arduino BLE 33 Sense
